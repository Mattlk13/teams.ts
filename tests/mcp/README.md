# Agent: MCP (Model Context Protocol)

A bot that acts as an MCP server.

## Run

```bash
$: npm run dev
```

# Run inspector

```bash
$: npm run inspect
```

To run on teams, run:

```bash
npx @microsoft/teams.cli config add ttk.basic
```

This will add all the ttk related configs.

Then run the sample via ttk.
